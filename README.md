OpenCircle is a PHP & MySQL based community website where users can
register, log in, and share posts in a common feed.

## Features
- User Registration & Login
- Session Management
- Create Posts
- Community Feed
- Secure Logout

## Tech Stack
- PHP
- MySQL
- HTML, CSS
- Bootstrap

## Purpose
This project was built to understand backend development,
authentication, and real-world web application flow.

## How to Run
1. Clone the repository
2. Import the SQL file into MySQL
3. Update database credentials in `config/db.php`
4. Run the project on XAMPP / WAMP
